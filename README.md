# ASP.NET MVC - Partial Views

Projeto destinado aos desenvolvedores ASP.NET que queiram conhecer o MVC, com a utilização das Partial Views, que são partes do código que podem compor suas Views, de forma que você desenvolve sua Partial View e pode aplicar esta em uma ou mais Views, conforme a necessidade do seu projeto, aproveitando o código e centralizando as informações e logicado seu negócio neste ponto.

Esta abordagem concede ao desenvolvedor maior produtividade no desenvolvimento de sistemas e facilita a manutenção do código relacionado a este ponto no seu projeto.
